# ☁️ Cloud File Storage System (Mini Google Drive)

A scalable cloud storage backend built using Java, Spring Boot, Spring Security, JWT, MySQL, and JPA that enables users to securely upload, organize, manage, and share files in a cloud environment.
The system is designed to provide a secure and efficient file management platform where users can store their documents, images, videos, and other digital assets. It supports folder-based organization, file sharing through secure links, storage usage tracking, and robust access control mechanisms. The application follows industry-standard security practices using JWT-based authentication, refresh token rotation, token blacklisting, and role-based authorization to ensure that users can access only their own resources.
Built with scalability and maintainability in mind, the project simulates the core functionalities of modern cloud storage platforms such as Google Drive and Dropbox while demonstrating real-world backend development concepts including file streaming, metadata management, hierarchical folder structures, storage quota management, secure sharing, and enterprise-grade authentication and authorization workflows.
